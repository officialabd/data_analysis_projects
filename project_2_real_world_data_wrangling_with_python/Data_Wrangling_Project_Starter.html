1. Gather data¶

1.1. Problem Statement¶

While searching for a dataset of Palestinian tech employees, I couldn't find any dataset that could help analyze the Palestinian labor market and tech experiences. So, I decided to collect some data using scraping methods on LinkedIn, and to address the privacy concerns I didn't collect any personal information. Moreover, to study the market, I collected another dataset of jobs with their skills, the importance of the skills, and other data to help analyze the market.

So, after merging these datasets, we can create visualizations to answer the following question:

  • How do the most in-demand technical skills vary between different job roles in the Palestinian tech labor market?

1.2. Datasets¶

Here are the available datasets:

  • Palestinian LinkedIn Profiles
  • Job Dataset

1.2.1 Dataset 1: Palestinian LinkedIn Profiles¶

The dataset comprises LinkedIn profiles of Palestinian professionals, capturing a wide array of information including work experience, education, certifications, skills, and more. This comprehensive dataset was collected to analyze the tech labor market in Palestine, identify emerging trends, and enrich the available labor market data.

Type: JSON Files

Method: The data was scraped from LinkedIn profiles using APIFY Linkedin profile scraper tool

Info: The data was of 59 JSON files since it was scraped using multiple bots. So, I merged them under 1 JSON file and removed any personal info to address any privacy concerns before we start digging here as shown below in the code snippet:

import pandas as pd
import os

folder_path = '/content/json'

files = [f for f in os.listdir(folder_path) if f.endswith('.json')]

columns_to_keep = [
    'id', 'jobTitle', 'occupation', 'headline', 'industryName',
    'geoCountryName', 'countryCode',
    'followersCount', 'connectionsCount', 'student',
    'companyName', 'companyPublicId', 'companyLinkedinUrl',
    'positions', 'educations', 'certifications', 'courses', 'honors', 'languages', 'skills', 'volunteerExperiences'
]

dataframes = []

for file in files:
    file_path = os.path.join(folder_path, file)
    df = pd.read_json(file_path)
    dataframes.append(df)

merged_df = pd.concat(dataframes, ignore_index=True)

df = merged_df[columns_to_keep]

def make_hashable(x):
    if isinstance(x, dict):
        return tuple((k, make_hashable(v)) for k, v in sorted(x.items()))
    elif isinstance(x, list):
        return tuple(make_hashable(i) for i in x)
    return x

for col in df.columns:
    df.loc[:, col] = df[col].apply(make_hashable)

df.to_json('out/palestinian_linkedin_profiles.json')

print('Merged and saved the DataFrame successfully.')

# Function to find the best match and similarity score
def get_best_match(skill, choices):
    global progress_counter
    global total
    progress_counter += 1
    print(f'Progress: {progress_counter}/{total}') if progress_counter % 100 == 0 else None

    # Convert skill to string
    skill = str(skill)

    # Find choices that contain the skill
    containing_choices = [choice for choice in choices if skill in choice]

    if containing_choices:
      try:
          # If there are multiple choices, find the best match among them
          match = process.extractOne(skill, containing_choices, scorer=fuzz.token_sort_ratio)
          if match:
              return match[0], match[1]
          else:
              return None, 0
      except Exception as e:
          print(f"Error processing skill '{skill}': {e}")
          return None, 0
    else:
      try:
          # If there are multiple choices, find the best match among them
          match = process.extractOne(skill, choices, scorer=fuzz.token_sort_ratio)
          if match:
              return match[0], match[1]
          else:
              return None, 0
      except Exception as e:
          print(f"Error processing skill '{skill}': {e}")
          return None, 0
